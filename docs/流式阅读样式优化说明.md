# 流式阅读页面样式优化说明

## 优化概述

基于 Anthropic 设计规范，对流式阅读页面进行了全面的视觉优化，采用 Poppins（标题）+ Lora（正文）字体组合，以及 Anthropic 品牌色系。

## 主要变更

### 1. 字体系统

- **标题字体**：Poppins（无衬线，现代感强）
  - 用于所有标题（h1-h6）
  - 字重：400（常规）、600（半粗）、700（粗体）
  
- **正文字体**：Lora（衬线，优雅易读）
  - 用于段落、列表、引用等正文内容
  - 支持斜体变体
  - 字重：400（常规）、500（中等）、600（半粗）

- **等宽字体**：Fira Code
  - 用于代码块和行内代码

### 2. 颜色系统

采用 Anthropic 品牌色板：

**主色调**
- `--color-dark: #141413` - 深色文字
- `--color-light: #faf9f5` - 浅色背景
- `--color-mid-gray: #b0aea5` - 中灰色
- `--color-light-gray: #e8e6dc` - 浅灰色

**强调色**
- `--color-orange: #d97757` - 主要强调色（标题下划线、表头）
- `--color-blue: #6a9bcc` - 链接色
- `--color-green: #788c5d` - 标签和成功状态

### 3. 排版优化

**标题层级**
- H1：48px，橙色下划线（3px）
- H2：36px，浅灰色下划线（2px），上边距 2.5em
- H3：28px，上边距 2em
- H4：22px，上边距 1.5em

**正文**
- 字号：18px
- 行高：1.8（提升可读性）
- 段落间距：1.5em

**列表**
- 行间距优化
- 使用 Lora 字体保持一致性

### 4. 组件样式

**引用块（Blockquote）**
- 左侧橙色边框（4px）
- 浅灰色背景
- 斜体样式
- 内边距优化

**代码块**
- 浅灰色背景
- 圆角设计
- 等宽字体
- 行内代码和块级代码区分明显

**表格**
- 橙色表头背景
- 白色表头文字
- 行悬停效果（浅灰色背景）
- 边框使用浅灰色

**链接**
- 默认蓝色
- 悬停时变为橙色
- 平滑过渡效果

### 5. 交互优化

- 表格行悬停效果
- 链接颜色过渡动画
- 滚动条样式优化（使用品牌色）

## 技术实现

### 文件修改清单

1. **app/globals.css**
   - 引入 Google Fonts（Poppins + Lora）
   - 定义 CSS 变量（颜色、字体）
   - 全局样式重置
   - 组件样式定义

2. **components/content-reader.tsx**
   - 更新所有 Markdown 组件样式
   - 应用品牌色和字体
   - 优化间距和排版

3. **app/page.tsx**
   - 更新背景色为品牌浅色

## 设计原则

1. **可读性优先**：使用 Lora 衬线字体提升长文本阅读体验
2. **层次分明**：通过字号、字重、颜色建立清晰的视觉层级
3. **品牌一致性**：严格遵循 Anthropic 设计规范
4. **优雅简洁**：避免过度装饰，保持专业感

## 浏览器兼容性

- 现代浏览器（Chrome、Firefox、Safari、Edge）完全支持
- Google Fonts 自动回退到系统字体
- CSS 变量在所有现代浏览器中支持良好
